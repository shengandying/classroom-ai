# 🏫 林晟课堂 AI 情感状态监测与全景热力图系统

本推演项目荣幸参加本次数据可视化大赛。系统实现了从端到端的完整闭环：从原始视频的自动化人脸清洗，到基于深度学习（ResNet V2）的模型微调，再到摄像头毫秒级时序日志捕获，最终通过 GCN 拓扑风格的热力图进行动态回放。

## 📁 仓库源码技术导览
* **01_File0_数据清洗**：接管原始视频，利用 OpenCV 自动识别人脸并空间裁剪，规范化 224x224 数据集。
* **02_File1_模型训练**：基于 ML.NET 引入 ResNet V2 50 层卷积神经网络，利用显卡（RTX 5060 Ti）加速微调，建立 4 分类情感基线模型。
* **03_File2_实时监控**：无缝接管外接摄像头，进行动态帧率识别，每 10 秒高精度捕捉一次情感快照，优雅落盘为结构化时间序列 CSV 矩阵。
* **根目录 (index.html)**：前端热力图大屏，支持通过时间滑动条（Slider）对 30 人班级的情感演变进行沙盘推演回放。
